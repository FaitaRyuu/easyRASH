Annotations

When you'll open a paper another tab will appear where you'll have all the annotations regarding that paper with different options per annotation:

delete_forever Remove

Completly remove an annotation, without the possibility to undo the change. You can remove only your annotations.

mode_edit Edit

Change an annotation. As for the remove you'll onlybe able to edit your annotations.

description View

When you'll click view the menu will close and you'll be redirected to the portion of the paper relative to that annotation. You'll still be able to read the annotation on the bottom of your screen.

account_circle User Menu

Change Account Information

Once you'll click the Edit button inside your user menu a new window will show. There you'll be able to chage some or all your account informations.

If you want to abort the changes just press the "esc" key or click outside the edit window.

If you want to confirm the changes, click Confirm and a pop up will show you if the changes will be done or not.

Review a Paper

To create an annotation or to insert a final decision to a paper you'll need to enter the ANNOTATOR mode by clicking this button on the top right of your screen near the user menu.

Once you'll enter ANNOTATOR mode the button will switch to READER and add will appear on the bottom right of your screen.

As a CHAIR you'll be able to:

message Create a new annotation

Before adding a new annotation you'll first have to select the part of the text relative to your annotation. A new window will show where you'll be able to add your annotation. Note that you can only annotate one paragraph at a time

done_all Insert final decision

As a chair you'll be able to decide if the paper is ACCEPTED or REJECTED.

save Save Everything is IMPORTANT!

All the changes are partial. To validate them permanently you'll need to save them all by clicking this button before exiting the ANNOTATOR mode.
As a REVIEWER you'll be able to:

message Create a new annotation

Before adding a new annotation you'll first have to select the part of the text relative to your annotation. A new window will show where you'll be able to add your annotation. Note that you can only annotate one paragraph at a time

done Insert overall judgment

Same as for the Chair you'll be able to chose if the paper is ACCEPTED or REJECTED but your decision as a Reviewer will not be definitive.

save Save Everything is IMPORTANT!

All the changes are partial. To validate them permanently you'll need to save them all by clicking this button before exiting the ANNOTATOR mode.
